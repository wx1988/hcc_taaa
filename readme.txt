To run the service, `python code.py 8080`. You could access the website at http://localhost:8080/ .

Setup on Windows environment is not supported for now.

Software required:
* Python: https://www.python.org/
* pymongo: mongdo database
* psycopg2: postgresql database,
* web.py: web framework, `easy_install web.py`


Optional:
* numpy, scipy: scientific computation
* sas7bdat: the data operation for sas database
* shapely: geometry operation
